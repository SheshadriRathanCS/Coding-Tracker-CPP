# Coding Tracker (C++)

A console-based Coding Tracker built using **C++**, **Object-Oriented Programming**, **STL**, and **File Handling**.

## Features

- Add new coding problems
- Delete existing problems
- Search problems by name
- Display all problems
- View statistics
- Track solved problems
- Track revision count
- Save data to file
- Load data automatically on startup

## Technologies Used

- C++
- Object-Oriented Programming
- STL (`vector`, `map`)
- File Handling
- Modular Programming

## Project Structure

Coding-Tracker-CPP

## Build

Compile:

```bash
g++ main.cpp Problem.cpp Tracker.cpp -o CodingTracker
```

Run on Windows:

```bash
.\CodingTracker.exe
```
